# supertyper

Uni programming project

## Getting started

1. Install `pipenv`

```bash
$ pip install pipenv
```

2. Install dependencies

```bash
$ pipenv install
```

3. Start pipenv shell

```bash
$ pipenv shell
```

4. Once the shell is started, run `reqtexts.py`

```bash
$ py reqtexts.py
```

5. Finally, run `__main__.py` to start game

```bash
$ py __main__.py
```
